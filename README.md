# XOLguesser
Python country guesser game made for a terminal. Featuring autocorrect, hints, continent based rounds, and basic stats.

# Download
To download, download the XOLguesser.py file attached. To execute in terminal, run python3 /home/user/Downloads/XOLguesser.py or diffrent if moved/downloaded it somewhere else.

# Gameplay
To play, just execute the file and follow all the onscreen directons. 

# Dependances
In order to play, you must have Python, which can be installed as such:

Debian/Ubuntu based systems: sudo apt install python3

Fedora/Red Hat based systems: sudo dnf install python3

Arch based systems: sudo pacman -S python3
